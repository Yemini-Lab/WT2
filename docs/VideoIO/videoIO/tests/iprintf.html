DESCRIPTION 

IPRINTF(...)
  Same as FPRINTF, but the current indentation level (as regulated by
  IENTER and IEXIT) is used to prepend whitespace to the beginning of each
  line.  A newline is always added to the end.

Example:
  ienter('foo...');
  iprintf('2+2=%d', 4);
  iexit('...foo');
